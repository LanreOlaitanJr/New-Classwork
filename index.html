Introduction to Git

Git is a Distributed Version Control System (DVC) that is used for tracking changes in sources code during software development. It allows multiple developers to collaborate on a project simultanously by providing mechanism for branching, merging and tracking changes.

GitHub is a web based platform built around GIT that helps developers store, track and collaborate on code project.

Repository oten abrriviated as "repo" in the context of GIT is essentially a storage locatiion where projects files and their complete version history are stored. It is like a folder tha contains all the files and directions accociated with projects along with the entire history of changes made to those files.
